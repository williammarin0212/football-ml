# FOOTBALL - ML


## Introducción

Análisis, modelamiento y despliegue de dashboard sobre analítica de fútbol de Inglaterra basado en datos que utiliza modelos de Machine Learning para predecir resultados de partidos y simular análisis tipo apuestas, mostrando probabilidades, métricas de rendimiento y visualizaciones interactivas a partir de datos históricos.

## Instrucciones para ejecutar los notebooks

1. Instalar las dependencias:

```bash
pip install -r requirements.txt
```

2. Abrir Jupyter Notebook:

```bash
jupyter notebook
```

3. Ejecutar los notebooks en este orden dentro de cada carpeta:

* `EDA(Análisis Exploratorio de Datos)`: análisis exploratorio de los datos
* `Modelo 1` y `Modelo 2` : entrenamiento del modelo, feature engineering y generación de predicciones

4. Ejecutar todas las celdas en orden.

**Nota:** No es necesario ejecutar los notebooks para usar el dashboard, ya que los datos ya están incluidos.
